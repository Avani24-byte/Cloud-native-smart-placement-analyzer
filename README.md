Smart Placement Analyzer is a web application that allows students to upload their resumes and instantly receive an ATS score, skill analysis, and job role recommendations powered by AWS. It uses Flask for the backend, AWS S3 for resume storage, and DynamoDB to save student results for future reference.

Current Features:
- Resume Upload
- AWS S3 Cloud Storage
- Resume Dashboard
- Student Information Collection
- DynamoDB Integration
- ATS Score Generation

Upcoming Features:
- Resume Parsing
- Job Recommendation Engine
